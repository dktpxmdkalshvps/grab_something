# 🎯 The "No" is Not an Option: Interactive Decision Maker

이 프로젝트는 답정너(답은 정해져 있고 너는 대답만 하면 돼) 스타일의 유쾌한 인터랙티브 웹 페이지 모음입니다. 사용자가 거절 버튼을 누르려 할수록 거절은 더 어려워지고, 수락은 더 쉬워지는(물리적으로) 특별한 경험을 선사합니다.

## ✨ 주요 기능 (Key Features)

* **도망가는 'No' 버튼**: 마우스 커서나 터치가 가까워지면 'No' 버튼이 계산된 궤적으로 도망갑니다.
* **성장하는 'Yes' 버튼**: 사용자가 거절하려고 시도할 때마다 'Yes' 버튼이 점점 커져 화면을 압도합니다.
* **화려한 피날레**: 'Yes'를 클릭하는 순간 `canvas-confetti` 라이브러리를 이용한 폭죽 효과와 함께 성공 화면이 나타납니다.
* **반응형 디자인**: 모바일 터치 이벤트와 데스크톱 마우스 이벤트를 모두 지원합니다.

---

## 📂 포함된 테마 (Included Themes)

### 1. 🍻 오늘 회식 갈래 말래? (`hoesig.html`)

'갈래말래' 밈을 활용한 버전입니다.

* **컨셉**: 피할 수 없는 팀 회식 권유
* **주요 특징**: 김원훈, 조진세의 코믹한 이미지 삽입, 역동적인 그라데이션 배경.
* **기술적 특징**: 0.15배씩 커지는 버튼 로직과 160px 감지 거리 적용.

### 2. 😋 Yummy! (`yummy_button.html`)

음식과 맛집 탐방을 주제로 한 글로벌 버전입니다.

* **컨셉**: "맛있는 거 먹으러 갈래?" 라는 거절할 수 없는 제안
* **주요 특징**: 따뜻한 오렌지/옐로우 톤의 색감, 깔끔한 이모지 아트.
* **기술적 특징**: 테마에 맞춘 난색 계열(Warm colors) 폭죽 효과 커스텀.

## 🚀 사용 방법 (How to Use)

1. 저장소의 파일을 다운로드합니다.
2. `hoesig.html` 또는 `yummy_button.html` 파일을 브라우저로 엽니다.
3. 친구, 동료, 혹은 연인에게 링크를 보내고 그들의 당황하는 모습을 감상하세요!

> **Note**: 이 코드는 순수 HTML/JS로 작성되어 별도의 서버 설정 없이 로컬에서 즉시 실행 가능합니다.
